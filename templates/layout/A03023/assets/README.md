# A03023 资源说明

本目录仅定义交付包中的资源位置，不附带真实或占位二进制资源。

- `models/`：放置 manifest 指定且人工检查过的 `.skp`；
- `references/`：放置已审核尺寸来源、SKU 台账等只读依据；
- 所有资源均使用项目包内相对路径，并在 `../handoff-manifest.json` 记录 SHA-256；
- 客户模型、图纸和台账不得提交到公共仓库；交付介质应实施最小权限和留存策略；
- 同名文件哈希不同视为不同资源，禁止静默替换或仅按文件名重连。

资源到位后仍需用户在 LayOut 中打开 `.layout`、刷新模型引用、检查关联尺寸并点击导出。
